Download node js from 
https://nodejs.org/dist/v14.16.0/node-v14.16.0-x64.msi
After download install it.

Open the code in visual studio code.
press CTRL and ~ <= ~ is placed before the number 1.

Terminal is open on the folder where project file are.

Type  (npm install)
All the dependancies will install

Then type (npm start)
Then project will open in local host 3000
